# UI Copy Kit — Recoverlution

Keep the voice: claim → human truth → inevitable outcome. Short, calm, verb-forward. No streaks, no shame, no hype.

## Sacred Phrases (always allowed)
- A quick return?
- Make a return
- Receipt of Return / Receipt of Repair / Receipt of Integrity
- Held in the wild
- Default forming
- Bring in a human
- Proof, without punishment
- Quiet by default. Consent by design.
- Humans when humans matter.

## Core Objects
- **Cue (NaviCue)**
  - Title: A quick return?
  - Body: 30 seconds. One move.
  - Why now (rotate): Heat is high. Short moves land best here. / You’ve returned this way before. We’re reinforcing it. / Evening drift is common. We’re routing earlier. / This is a repair moment. We’re sealing proof.
  - CTA: Make a return
  - Tertiary (only when needed): Bring in a human
- **Move**
  - Title (verb): Breathe low. / Name the urge. / Change the room. / Send one truth.
  - Body: 30 seconds. / One step. / 3 minutes.
  - Why this/why now: Heat is high. We’re keeping it small. / This is how the week gets steadier.
  - Completion CTA: Seal the receipt
- **Receipt**
  - Title: Receipt of Return (or Repair/Integrity)
  - Fact: Returned in 48 seconds. / Held under pressure. / Repair completed.
  - Actions: Replay · Keep
- **Proof strip**
  - Nodes: Receipt → Transfer → Trajectory
  - Micro line: Default forming.
- **Return history (MTTR)**
  - Header: Return.
  - Sub: The skill that changes everything.
  - Lines: Return is getting faster. / You’re coming back sooner. / Baseline is rising.
- **Resistance moment**
  - Title: Keep it simple?
  - Body: One move is enough.
  - CTA: Make a return
  - Secondary: Not now
  - If “Not now”: That’s data. We’ll reroute.
- **Escalation**
  - Title: Bring in a human.
  - Body: This moment deserves care.
  - Primary: Contact support
  - Secondary: Ground for 30 seconds
  - Grounding step: Feet. Breath. Room. / Name 3 things you can see.
- **Connector consent**
  - Title: Connect real life.
  - Sub: Context isn’t surveillance. It’s timing.
  - Purpose options: Better timing / Prep before moments / Proof of transfer
  - Quiet hours toggle: Keep it quiet — Support that respects your day.
- **Support graph**
  - Title: Your people.
  - Role prompt: Who are they to you? (Witness / Peer / Safety / Clinician)
  - Visibility: What can they see? (Nothing / Only requests / Receipts you choose)
  - Safety: When things get heavy — Allow emergency contact (only when you choose, or when safety rules apply).

## Empty States (kind)
- Nothing to prove today. That’s allowed.
- Keep it simple. One move is enough.
- Not now is data. We’ll reroute.

## Notifications (template)
- A quick return? — 30 seconds. One move.
- Never: don’t forget / keep your streak / you haven’t logged / guilt.

## Banned Language (UI)
- Streak, engagement, points, gamified, leaderboard, compliance, tracking, submit, complete, finish, warning, failure.

## Tone Rules
- Short lines. One idea per line. Verbs over nouns. No praise/shame. Explainability in one line: Why this, why now.

## State Guardrails
- CALM: options OK; reflection allowed; longer microcopy fine.
- HEAT: single CTA; minimal language; short move; no menus.
- FRACTURE: remove choice; human-first; grounding; no alarms.
- REPAIR: seal proof; gentle next step; offer replay/keep.
